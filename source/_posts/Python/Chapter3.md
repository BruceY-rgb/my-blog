## 2. 赋值语句

### 2.1 基本概念

在 Python 中，赋值操作实现了将一个名字（变量）绑定到一个对象上。

```python
# 基本语法格式：变量 = 对象
x = 1
y = 2
```

### 2.2 赋值语句形式

| 形式 | 示例 | 说明 |
|------|------|------|
| **单变量赋值** | `x = 1` | 最基本的形式 |
| **多重赋值** | `x, y = 4, 8` | 同时给多个变量赋值 |
| **链式赋值** | `x = y = z = 1` | 操作顺序是从右向左进行 |
| **序列解包** | `a, *b = "1234"` | 将可迭代对象解包赋值 |
| **增强赋值** | `j += 3` | 相当于 `j = j + 3` |

```python
# 单变量赋值
x = 1
y = 2

# 多重赋值
x, y = 4, 8

# 链式赋值
x = y = z = 1

# 序列解包
a, *b = "1234"     # a='1', b=['2', '3', '4']
*a, b = "1234"      # a=['1', '2', '3'], b='4'
a, *b, c = "1234"   # a='1', b=['2', '3'], c='4'

# 增强（复合）赋值
j = 5
j += 3              # j = j + 3 = 8
k /= j + 2          # k = k / (j + 2)

# 字符串拼接
s = "hello"
s += "world"        # s = "helloworld"
```

### 2.3 例题：交换两个值

**任务**：输入两个整数 a 和 b，并将两个数交换后输出

```python
# Python 实现方式 - 使用多重赋值
a = int(input())
b = int(input())
a, b = b, a  # 仅需一行代码即可交换
print(a, b)
```

### 2.4 例题：计算三角形面积

**任务**：输入三角形的三边长度，求这个三角形的面积

```python
import math

a = int(input())
b = int(input())
c = int(input())

p = (a + b + c) / 2
area = math.sqrt(p * (p - a) * (p - b) * (p - c))

print("三角形的边长:{:3d},{:3d}, {:3d}".format(a, b, c))
print("三角形的面积:{:8.3f}".format(area))
```

## 3. 条件判断

### 3.1 布尔表达式

条件判断是程序的"决策大脑"，决定了代码执行的路径。

- 通常采用布尔表达式，也可以直接对对象进行真值测试
- 条件判断的结果是 True 或 False，程序根据结果执行相应的流程

### 3.2 关系运算

关系运算是用于比较两个值的关系（如大小、相等性），该运算会返回一个布尔值。

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| == | 判断两个操作数是否相等 | 3 == 2 | False |
| != | 判断两个操作数是否不相等 | 3 != 2 | True |
| > | 判断左操作数是否大于右操作数 | 3 > 2 | True |
| < | 判断左操作数是否小于右操作数 | 3 < 2 | False |
| >= | 判断左操作数是否大于或等于右操作数 | 3 >= 2 | True |
| <= | 判断左操作数是否小于或等于右操作数 | 3 <= 2 | False |

!!! warning
数据类型要求类型一致
!!!

**链式比较**：
```python
a < b < c  # 等价于 (a < b) and (b < c)
```

### 3.3 逻辑运算

#### 真值表

| 逻辑量1 | 逻辑量2 | and 结果 | or 结果 |
|---------|---------|----------|---------|
| False | False | False | False |
| False | True | False | True |
| True | False | False | True |
| True | True | True | True |

| 逻辑量 | not 结果 |
|---------|----------|
| False | True |
| True | False |

#### 运算符优先级

```
not(非) > and(与) > or(或)
```

可以使用括号明确指定运算顺序。

#### 示例：闰年判断

```python
# 闰年的条件：年份能被4整除，且年份不能被100整除，或者年份能被400整除
year = int(input())
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
```

#### 短路原则

- **and 运算符**：当第一个操作数为假时，第一个操作数的值就是表达式的结果，不计算第二个操作数
- **or 运算符**：当第一个操作数为真时，第一个操作数的值就是表达式的结果，不计算第二个操作数

```python
# 示例：用户登入欢迎信息
name = input("Enter your name: ") or "Guest"
print(f"Welcome, {name}!")
```

### 3.4 成员运算和身份运算

#### 成员运算符

用于检查某个值是否存在于容器中。

| 运算符 | 功能 | 示例 | 结果 |
|--------|------|------|------|
| in | 判断元素是否存在于容器中 | 2 in [1, 2, 3] | True |
| not in | 判断元素是否不在容器中 | 'ab' not in 'bac' | True |

#### 身份运算符

比较两个对象的身份，通常就是指内存地址。

| 运算符 | 功能 |
|--------|------|
| is | 判断两个对象是否是同一对象（内存地址是否相同） |
| is not | 判断两个对象是否不是同一对象 |

!!! tip
- `==` 比较的是**值**是否相等
- `is` 比较的是**身份**（内存地址）是否相同

> 这里存在一个字面量缓存机制的问题，`[-128,127]`范围内的数
> `a = 100`
> `b = 100`
> `a is b # true`
!!!

### 3.5 真值测试

Python 在条件判断（if, while）或逻辑运算中，自动将任意对象转换为布尔值的机制。

| 类别 | 假值 (Falsy) → False | 真值 (Truthy) → True |
|------|----------------------|---------------------|
| 数值 | 0, 0.0, 0j（所有零） | 所有非零数值（如 1, -5, 3.14） |
| 序列/集合 | ""（空串）, [], (), {}, set() | 所有非空容器 |
| 特殊对象 | None, False | 所有其他对象（包括自定义类实例） |

## 4. 分支语句

### 4.1 单分支语句

**语法格式**：
```python
if 条件:
    语句块1
```

**流程图**：
```
条件
True → 语句块1
False → (跳过语句块)
```

**语法说明**：
- 条件：进行真值测试，结果为 True 或者 False
- 冒号（:）：if 语句行末**必须跟冒号**
- 缩进：if 语句下方的语句块必须向右缩进，通常是 4 个空格或 1 个 Tab
- 语句块：可以是一条语句，也可以是多条语句

!!! warning
注意缩进的重要性！不同的缩进会导致完全不同的程序逻辑。
!!!

### 4.2 二分支语句

**语法格式**：
```python
if 条件:
    语句块1
else:
    语句块2
```

**流程图**：
```
条件
True → 语句块1
False → 语句块2
```

**语法说明**：
- if 和 else 必须严格左对齐
- 冒号（:）：if 语句行末以及 else 行末必须跟冒号
- 缩进：语句块 1、2 必须向右统一缩进
- 取消缩进：当编写不属于这个双分支的后续代码时，将光标移回与 if/else 相同的垂直位置

### 4.3 多分支语句（if-elif-else）

**语法格式**：
```python
if 条件1:
    语句块1
elif 条件2:
    语句块2
...
elif 条件n:
    语句块n
else:
    语句块n+1
```

**语法说明**：
- if、elif、else 必须严格左对齐，行末必须跟冒号
- 每个语句块都必须向右统一缩进
- else 不是必须的，但如果写就必须放在最后

### 4.4 例题：分段函数

```python
# 例：分段函数
x = int(input())

if x < 0:
    f = -1
elif x == 0:
    f = 0
else:
    f = 2 * x

print(f)
```

### 4.5 课堂练习：计算阶梯水费

**任务**：
- 当 x 不超过 15 吨时，y = 4x/3
- 超过后，y = 2.5x - 17.5
- 输出结果小数部分保留 2 位

```python
x = float(input())

if x <= 15:
    y = 4 * x / 3
else:
    y = 2.5 * x - 17.5

print(f"{y:.2f}")
```

### 4.6 课后练习：求分段函数

**任务**：输入 x，求分段函数 f(x) 的值，保留 2 位小数

$$
f(x) = \begin{cases}
e^x & x \leq 1 \\
x^2 - 1 & x > 1
\end{cases}
$$

```python
import math

x = float(input())

if x <= 1:
    y = math.exp(x)
else:
    y = x ** 2 - 1

print(f"{y:.2f}")
```

### 4.7 嵌套的分支语句

**语法格式**：
```python
if 条件1:
    if 条件2:
        语句块1
    else:
        语句块2
elif 条件3:
    if 条件4:
        语句块3
    else:
        语句块4
```

!!! warning
在使用嵌套结构时：
- 不同缩进代表了不同级别语句块
- else 总是与同一缩进级别、最近的 if 匹配
- 应尽量避免层级过深（通常不超过 3 层），否则会导致代码难以阅读和维护
!!!

### 4.8 条件表达式

条件表达式类似 if-else 语句，是用来直接得到值。条件表达式是三元的，需要三个值。

```python
# 语法格式：值1 if 条件 else 值2
y = 10 if x > 20 else 30
# 当 x 大于 20 时，y = 10，否则 y = 30
```

**使用场景**：
- 简单二选一赋值
- 保持代码简洁的表达式
- 不建议在多层嵌套的场景使用

## 5. 循环语句

### 5.1 循环语句类型

循环语句用于重复执行某段代码，是控制流程的重要组成部分。

| 类型 | 说明 |
|------|------|
| **条件控制型 while** | 根据一个逻辑条件（循环条件），在条件成立时执行循环体语句，不成立时结束循环 |
| **遍历型 for** | 从序列（或其他可迭代对象）中每次取一个元素，然后执行循环体，直到全部元素取完 |

**循环语句的作用**：
- 自动化处理：实现批量数据操作的核心机制
- 迭代优化：例如在 AI 模型训练中，循环结构是不可或缺的基础支撑
- 资源控制：有效防止无限循环，避免系统资源的无谓消耗

### 5.2 while 循环

**语法格式**：
```python
while 条件:
    语句块
```

**执行流程**：
1. 判断条件是否成立
2. 如果条件成立，则执行语句块
3. 回到第 1 步
4. 如果条件不成立，结束循环

!!! warning
- 循环语句书写必须缩进
- 在循环体内部，应该有改变循环条件的语句，以控制循环的次数，避免产生无限循环（死循环）
!!!

### 5.3 例：计算存款年数

**任务**：某客户在银行中存入 1000 元人民币，年利率为 3%，假设该客户每年的利息都不取出，计算至少存几年本息合计达 1500 元。

**本息计算公式**：$a = p(1 + r)^y$

```python
p = 1000      # 本金
r = 0.03      # 年利率
a = 1500      # 目标本息
y = 0         # 年数

while True:
    y += 1
    total = p * (1 + r) ** y
    if total >= a:
        print(f"需要存 {y} 年")
        print(f"本息合计: {total:.2f} 元")
        break
```

### 5.4 for 语句

**语法格式**：
```python
for 变量 in 可迭代对象:
    语句块
```

**执行过程**：
1. 变量先被赋给可迭代对象的第一个值，并执行语句块
2. 然后变量被赋给可迭代对象中的第二个值，再次执行语句块
3. 该过程一直继续，直到穷尽这个可迭代对象中的所有元素

### 5.5 range 对象

range 是 Python 内置的类型，它可以认为是一个不可变的数字序列，通常用在 for 循环。

```python
range(start, stop, step)
```

| 参数 | 说明 |
|------|------|
| start | 计数从 start 开始。默认是从 0 开始。例如 range(5) 等价于 range(0, 5) |
| stop | 计数到 stop 结束，但不包括 stop。例如：range(0, 5) 是 0, 1, 2, 3, 4，不包括 5 |
| step | 步长，默认为 1。例如：range(0, 5) 等价于 range(0, 5, 1) |

!!! warning
三个值都是整型，取数是**左闭右开**
!!!

```python
range(5)           # 0, 1, 2, 3, 4
range(1, 6)        # 1, 2, 3, 4, 5
range(0, 10, 2)    # 0, 2, 4, 6, 8
range(5, 0, -1)    # 5, 4, 3, 2, 1
```

### 5.9 嵌套循环

与分支结构类似，循环结构也可以进行嵌套，即在一个循环的内部包含另一个或多个循环。

**语法格式**：
```python
for 变量_外 in 可迭代对象:    # 外层循环，控制整体轮次
    for 变量_内 in 可迭代对象:  # 内层循环：控制每轮次内的操作
        代码块
```

### 5.10 例：安全检测

**任务**：一个机器人需要检查一个 5×5 的方形区域（坐标范围：0≤x<5, 0≤y<5），若某个坐标点满足 x + y ≤ 4，则认为该点是安全的。

```python
for x in range(5):
    for y in range(5):
        if x + y <= 4:
            print(f"({x},{y}) 安全")
        else:
            print(f"({x},{y}) 危险")
```

## 6. break 和 continue

在循环结构中，需要更精细地控制程序的执行流程：

- **break**：在特定条件满足时提前终止整个循环
- **continue**：跳过当前迭代的剩余部分，直接进入下一次循环

| 关键字 | 作用 |
|--------|------|
| **break** | 用于立即终止当前所在的循环结构。循环体中剩余的代码不再执行，直接跳转循环之外的第一条语句。如果是嵌套循环，break 仅跳出其所在的那一层循环 |
| **continue** | 与 break 不同，continue 不会终止整个循环。程序会立即跳过当前循环体中剩余的代码，直接进入下一次循环的条件判断（while）或获取下一个元素（for） |

### 6.1 例：猜数游戏

```python
import random

secret = random.randint(1, 100)

while True:
    guess = input("请输入猜测的数字（输入 'q' 退出）: ")

    if guess == 'q':
        print("退出游戏")
        break

    if not guess.isdigit():
        print("请输入有效数字")
        continue

    guess = int(guess)

    if guess == secret:
        print("恭喜你，猜对了！")
        break
    elif guess < secret:
        print("太小了")
    else:
        print("太大了")
```

## 7. else 子句

### 7.1 while 和 for 循环的 else 子句

- 当循环正常结束（while 条件变为 False 或 for 完成全部遍历），执行 else 中的代码
- 当循环在执行过程中被 break 提前终止，则会跳过 else，直接执行循环之后的语句

**语法结构**：
```python
while 条件:
    if 条件:
        break
else:
    语句块2  # 循环正常结束时执行

for 变量 in 可迭代对象:
    if 条件:
        break
else:
    语句块  # 循环正常结束时执行
```

!!! tip
else 子句可以帮助减少代码中使用额外的标志变量来判断循环的退出条件
!!!

### 7.2 例：查找字符

**任务**：用户输入了一段文字和特定字符，查找该字符是否存在，存在则输出首次出现的位置，否则输出不存在。

```python
text = input("请输入一段文字: ")
char = input("请输入要查找的字符: ")

for i, c in enumerate(text):
    if c == char:
        print(f"字符 '{char}' 首次出现在位置 {i}")
        break
else:
    print(f"字符 '{char}' 不存在于文本中")
```

### 7.3 例：素数判断

**任务**：输入一个大于等于 2 的正整数，判断是否为素数。

```python
num = int(input("请输入一个大于等于 2 的正整数: "))

for i in range(2, int(num ** 0.5) + 1):
    if num % i == 0:
        print(f"{num} 不是素数")
        break
else:
    print(f"{num} 是素数")
```

!!! tip
数学优化：其实不需要试到 `num - 1`，只要试到 $\sqrt{num}$ 就可以了。如果在这个范围内没找到因子，后面也不可能有。
!!!
