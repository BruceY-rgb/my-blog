BruceY's Blog 使用说明 - 添加笔记完整步骤

  系统概述

  您的博客是基于 Hexo 静态网站生成器构建的，使用 Butterfly 主题。笔记以 Markdown
  格式存储在 source/_posts/ 目录下。

  添加笔记的完整步骤

  方法一：使用 Hexo 命令行工具（推荐）

  1. 创建新文章
  # 在项目根目录下执行
  hexo new "您的文章标题"
  这会在 source/_posts/ 目录下创建一个新的 Markdown 文件，文件名自动转换为英文。

  2. 编辑文章内容
  使用任意文本编辑器或 Markdown 编辑器打开生成的文件，编辑内容。
  3. 本地预览
  # 启动本地服务器预览
  hexo server
  访问 http://localhost:4000 查看效果

  4. 生成静态文件
  # 构建网站
  hexo generate

  5. 部署（可选）
  # 部署到服务器
  hexo deploy

  方法二：手动创建文件

  1. 直接创建文件
  在 source/_posts/ 目录下创建新的 .md 文件
  2. 文件命名规则
  文件名将成为文章的 URL，建议使用有意义的英文名称：
  - ✅ 正确：my-first-note.md、learning-javascript.md
  - ❌ 避免：中文文件名、空格过多的文件名

  ### Front Matter 参数说明

  | 参数          | 说明         | 示例                                   |
  | ------------- | ------------ | -------------------------------------- |
  | `title`       | 文章标题     | `title: 我的第一篇笔记`                |
  | `date`        | 创建日期     | `date: 2024-02-09 12:30:00`            |
  | `tags`        | 标签（数组） | `tags: [技术, 学习]`                   |
  | `categories`  | 分类（数组） | `categories: [前端开发]`               |
  | `description` | 文章描述     | `description: 这是一篇关于...`         |
  | `cover`       | 封面图片     | `cover: https://example.com/image.jpg` |

  ### 常用 Markdown 语法

  - **标题**：`# 一级标题`, `## 二级标题`, `### 三级标题`
  - **加粗**：`**粗体文本**`
  - **斜体**：`*斜体文本*`
  - **链接**：`[链接文字](URL)`
  - **图片**：`![图片描述](图片URL)`
  - **代码块**：\`\`\`语言类型
  - **行内代码**：`\`代码\``
  - **引用**：`> 引用内容`
  - **列表**：`- 列表项` 或 `1. 列表项`

  ### 文章管理建议

  1. **分类管理**
     - 使用有意义的分类名称
     - 一篇文章可以属于多个分类

  2. **标签使用**
     - 标签用于更细粒度的分类
     - 一篇文章可以有多个标签

  3. **封面图片**
     - 建议使用 1200x630 像素的图片
     - 可以是本地图片路径或网络图片 URL

  4. **摘要设置**
     - 在文章中添加 `<!-- more -->` 手动控制首页摘要长度
     - 摘要前的内容会显示在首页
